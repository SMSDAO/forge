# User Guide

## Getting Started

1. Navigate to the application URL
2. Click **Sign In** or **Create Account**
3. After login, you'll land on the **Dashboard**

## Navigation

The navigation bar at the top (desktop) or bottom (mobile) provides access to:

| Tab | Description |
|-----|-------------|
| Home | Landing page and quick start |
| Dashboard | Your personal overview |
| Users | User directory (admin access) |
| Admin | Administration panel (admin access) |
| Developer | Developer tools (developer access) |
| Settings | Account preferences |
| Docs | Documentation |

## Dashboard

Your dashboard shows:
- **Overview cards**: projects, deployments, API calls, uptime
- **Activity chart**: deployments and API calls over the last 7 days
- **Notifications**: recent alerts and system messages
- **Metered Usage**: current consumption vs. plan limits

## Settings

Access via **Settings** tab or user avatar menu.

- **Profile**: Update name, email, avatar
- **Notifications**: Configure email and deploy alerts
- **Security**: Enable 2FA, change password
- **Billing**: View plan and usage, upgrade subscription
- **API Keys**: Generate and manage personal API keys

## IDE

Access the integrated development environment at `/ide` to build and edit projects with AI assistance.
